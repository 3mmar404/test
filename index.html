التفاصيل الكاملة:

تركيب وتأسيس الموقع

  • إعداد الاستضافة وربطها بالدومين
  • تثبيت WordPress وضبط الإعدادات الأساسية
  • تنصيب قالب احترافي
  • ضبط اتجاه RTL ولغة الموقع

إعداد WooCommerce

  • تفعيل نظام المتجر
  • إعداد ضريبة، شحن، كوبونات، سياسات
  • تفعيل الحسابات والتسجيل

  • ربط بوابة (فوري)
  • إعداد API + Webhook
  • اختبار الدفع الحقيقي وضمان التشغيل

تصميم صفحات المتجر

  • تصميم احترافي للـ Homepage
  • صفحات: "من نحن" – "تواصل معنا" – "الشروط والسياسات"
  • صفحة المتجر – عرض المنتجات – تسجيل الدخول – حسابي – الطلبات... إلخ

رفع المنتجات

  • إضافة كافة الأصناف المتوفرة حاليًا يدويًا (اسم، سعر، وصف، صور...)
  • تصنيف المنتجات داخل المتجر
  • ضبط خيارات المقاسات / الألوان إن وُجدت

تحسين الأداء والحماية

  • حماية من الاختراق والسبام
  • تسريع الموقع عبر Cache + Optimizer
  • ضبط SEO باستخدام RankMath / Yoast
  • تركيب دعم مباشر (واتساب + Live Chat)

التهيئة لمحركات البحث

  • ربط Google Analytics + Search Console
  • رفع Sitemap وفهرسة أولية
  • إعداد العناوين والوصف الميتا

تسليم كامل

  • تجربة الشراء كاملة (من الدخول حتى الدفع)
  • تسليم لوحة تحكم قابلة للتعديل
  • إمكانية التعديل والإضافة لاحقًا من طرف العميل
